    fluid-block

    @mixin fluid-block($alignH: 'center', $alignV: 'center', $inline: false) { ... }

    Description

    Transforms containing selector into a container that can freely align child inline-blocks with no fixed width/height requirements. Beware of whitespace between inline elements.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $alignH

    Horizontal alignment of child elements (i.e. left, center, right). Option to specify left/right paddings respectively.

    		List'center'
    $alignV

    Vertical alignment of child elements (i.e. top, center, bototm). Option to specify top/bottom paddings respectively.

    		List'center'
    $inline

    Specifies if this block is an inline block.

    		Stringfalse

    screen

    @mixin screen($screen, $exclusive: false, $invert: false, $measurement: width, $touch: false) { ... }

    Description

    Media query for the minimum width/height threshold of the specified screen type.

    Parameters

    parameterNameparameterDescriptionparameterTypeparameterDefault value
    $screen

    Screen type (supports 'mobile', 'phablet', 'tablet', or 'desktop').

    		String—none
    $exclusive

    Specifies whether the media queries should be exclusive to this screen type. If false, the generated media query will use the minimum threshold of the selected screen type, meaning that bigger screen types will apply. If true, the generated media query will be bounded by the minimum and the maximum thresholds, inclusive.

    		Booleanfalse
    $invert

    Inverts the generated media query.

    		Booleanfalse
    $measurement

    Specifies width or height measurement to be used for the media queries.

    		Stringwidth
    $touch

    Specifues whether the condition is only valid for touch devices (requires Modernizr feature detection), namely the '.touch' class. If false (default), it does NOT use the '.no-touch' class, but instead applies content normally.

    		Booleanfalse

    Content

    This mixin allows extra content to be passed (through the @content directive).

    Requires

    • [mixin] touch
